<task type="auto">
  <name>Task 1: Define color palette with glass morphism support</name>
  <files>src/app/globals.css</files>
  <action>
Update globals.css to define CSS custom properties for the design system. Use Tailwind v4's CSS-first approach with @theme directive.

Create a color system that supports:
1. **Primary accent colors** - Bold, vibrant colors for interactive elements
2. **Glass morphism colors** - Semi-transparent backgrounds with blur-compatible tints
3. **Text colors** - High contrast for readability (Notion-like comfort)
4. **Surface colors** - Layered backgrounds for depth

Example structure:
```css
@import "tailwindcss";

@theme {
  /* Primary accent - bold, modern */
  --color-primary-500: #6366f1;
  --color-primary-600: #4f46e5;

  /* Glass morphism backgrounds */
  --color-glass-light: rgba(255, 255, 255, 0.7);
  --color-glass-dark: rgba(0, 0, 0, 0.3);
  --color-glass-border: rgba(255, 255, 255, 0.2);

  /* Text - high contrast for readability */
  --color-text-primary: #1a1a2e;
  --color-text-secondary: #4a4a68;
  --color-text-muted: #8888a0;

  /* Surfaces - layered depth */
  --color-surface-0: #f8f9fc;
  --color-surface-1: #ffffff;
  --color-surface-2: #f0f2f8;

  /* Backdrop blur for glass effect */
  --blur-glass: 12px;
}
```

Include both light mode defaults. Dark mode will be added in 02-04.
  </action>
  <verify>Tailwind compiles without errors, CSS variables accessible in browser devtools</verify>
  <done>Color palette defined with primary, glass, text, and surface tokens. Variables accessible globally.</done>
</task>
